# Neo-Controller-Lamp

---

Neo Controller adalah sebuah project IoT sederhana yang berfungsi untuk mengontrol lampu dengan modul ESP32-C3 secara wireless dengan menggunakan koneksi WiFi.

---

**Features**

* **Wireless Lamp controller:** Mengontrol lampu secara wireless.
* **Brightness controll:** Mengatur kecerahan lampu sesuai keinginan dan kebutuhan.
* **Low Energy Consumption:** Menggunakan board super-mini sehingga menghemat konsumsi daya.
* **Web-Based:** Berbasis website sehingga pengguna tidak perlu menginstall aplikasi apapun.
* **Quick Setup:** Pengguna hanya perlu mengatur ssid dan password sesuai dengan jaringan pengguna

---

**Usage**

1. Clone repositori ini
2. Buka file lamp-controller.ino di Arduino IDE
3. Atur `ssid` dan `password` WiFi yang sesuai dengan jaringan milik kamu
4. Upload program dan tunggu hingga terhubung
5. Buka browser dan ketik "http://172.25.193.244/"

---

**Components**

* Esp32-C3
* Led 3v
* Resistor 220Ω-1kΩ
